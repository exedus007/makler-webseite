FAQ

Häufige Fragen

Wie läuft eine Anfrage ab?

Sie senden Ihre Anfrage direkt über das Objekt-Popup. Die Objekt-ID wird automatisch mitgeschickt, damit Ihre Anfrage direkt korrekt zugeordnet ist.

Kann ich das Exposé online ansehen?

Ja. Wenn für ein Objekt ein Exposé hinterlegt ist, können Sie es direkt im Popup öffnen oder herunterladen.

Werden reservierte Objekte weiterhin angezeigt?

Ja. Reservierte Objekte bleiben sichtbar und sind klar gekennzeichnet. Nur verkaufte Objekte werden automatisch ausgeblendet.
